Step 1 – Unzip Your Blog

  1. Download the file I gave you: health_blog_starter.zip

  2. Extract it to a folder on your computer (e.g., health_blog_starter).

Step 2 – Create a GitHub Repository

  1. Go to https://github.com and log in (or create an account).

  2. Click the + (top-right) → New repository.

  3. Name it something like health-blog.

  4. Keep Public selected.

  5. Don’t add a README (you can, but not required).

  6. Click Create repository.

Step 3 – Upload Your Files

  1. In your new repo, click "Add file" → "Upload files".

  2. Drag & drop all the extracted files and folders (index.html, styles.css, script.js, posts/, assets/, etc.).

  3. Scroll down and click Commit changes.

Step 4 – Enable GitHub Pages

  1. In the repo, go to Settings → Pages (left menu).

  2. Under "Branch", select main (or master) and / (root).

  3. Click Save.

  4. Wait about 1–2 minutes — GitHub will give you a live link like:

    arduino
    https://your-username.github.io/health-blog/

Step 5 – Test Your Site

  • Open the link and see your health blog live.

  • Share the link with anyone!

💡 Extra Tip:
If you want the site to appear at https://yourname.github.io, simply name the repo yourname.github.io and repeat the steps.
